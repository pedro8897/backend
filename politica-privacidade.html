Política de Privacidade

Última atualização: 15/07/2026

Esta Política de Privacidade explica como o Kintsugi Sushi coleta, usa e protege os dados informados pelos clientes no site de delivery.

1. Dados que coletamos

Para permitir cadastro, atendimento e entrega dos pedidos, podemos coletar:

  • nome;
  • e-mail usado no login;
  • telefone para contato;
  • endereço de entrega, quando o pedido for para entrega;
  • itens do pedido, adicionais, total e forma de pagamento;
  • histórico e status dos pedidos.

2. Para que usamos os dados

Usamos os dados para:

  • criar e acessar a conta do cliente;
  • registrar, preparar e entregar pedidos;
  • enviar o pedido ao WhatsApp do restaurante;
  • informar o status do pedido ao cliente;
  • prestar atendimento e resolver dúvidas sobre pedidos.

3. Onde os dados ficam armazenados

Os dados de cadastro e pedidos ficam armazenados no Supabase, serviço usado pelo site para login, banco de dados e acompanhamento de pedidos. A senha do cliente é gerenciada pelo sistema de autenticação do Supabase e não é salva diretamente no código do site.

4. Uso do WhatsApp

Ao finalizar um pedido, o site abre o WhatsApp com a mensagem do pedido pronta para envio ao restaurante. Essa mensagem pode conter nome, telefone, endereço, itens do pedido, total e forma de pagamento.

5. Compartilhamento de dados

Os dados são usados para atendimento do próprio pedido. Não vendemos dados de clientes. Podemos usar serviços necessários para o funcionamento do site, como Supabase e WhatsApp, sempre relacionados ao cadastro, atendimento e entrega.

6. Direitos do cliente

O cliente pode solicitar acesso, correção ou exclusão dos seus dados pessoais. Para isso, entre em contato pelo WhatsApp do restaurante: 69 99345-6791.

7. Segurança

Adotamos medidas técnicas compatíveis com um site de delivery, como autenticação de clientes, armazenamento em serviço especializado e uso de chave pública para conexão com o banco de dados. Ainda assim, nenhum sistema é totalmente livre de riscos.

8. Atualizações desta política

Esta política pode ser atualizada para refletir mudanças no site, no atendimento ou em exigências legais. A versão mais recente ficará disponível nesta página.
